IMAGE SETUP INSTRUCTIONS
========================

To add the contract image to this module:

1. FOR MENU ICON (Custom Image):
   - Save your contract image as: static/img/contract_icon.png
   - Update contracts_menu_view.xml line 17 to use: icon="/hr_penalty2/static/img/contract_icon.png"
   - Currently using FontAwesome icon "fa-handshake-o" (handshake icon)

2. FOR MODULE DESCRIPTION ICON:
   - Save your image as: static/description/icon.png
   - This will appear in the Apps list

3. RECOMMENDED IMAGE SPECS:
   - Format: PNG
   - Size: 128x128 pixels or larger (square)
   - Transparent background preferred

Current setup:
- Menu uses FontAwesome handshake icon (fa-handshake-o)
- You can replace this with your custom image if preferred
